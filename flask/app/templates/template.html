Displaying a Template

Templates in Flask using Jinja2

Statements

{% if my_variable %}

This paragraph won't be displayed in the template because the variable "my_variable" does not exist. The if statement therefore returns False.

{% else %}

This paragraph will be displayed.

{% endif %}

Variables

{{ my_variable }}

The variable above will not be displayed because it doesn't exist!

Comments

{% if True %}

This will be displayed because the if statement returns True.

{% endif %} {# {% if True %}

This will not be displayed because it is commented out using Jinja2 syntax.

{% endif %} #}
